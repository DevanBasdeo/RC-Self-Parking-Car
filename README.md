# RC-Self-Parking-Car

Final Year Project – Autonomous Parking System

## Description
This project implements a sensor-driven autonomous parking system using an Arduino-based platform. The system detects parking gaps and executes a parallel parking manoeuvre using ultrasonic sensors and time-based control.

## Features
- Gap detection using ultrasonic sensors
- Autonomous parking sequence
- Bluetooth remote control (HC-05)
- Safety stopping using rear sensor and IR sensor

## Hardware
- Arduino Uno R4 WiFi
- HC-SR04 Ultrasonic Sensors
- IR Sensor
- L298N Motor Driver
- PCA9685 Servo Driver
- LM2596 Buck Converter
